# Функционал приложения

## Описание
Коротко: канбан-борда для управления задачами.

## Текущий функционал
- Борда на весь экран приложения
- Отключен дефолтный скролл страницы
- Дефолтный столбик слева: Inbox
- Inbox визуально выделен стилями
- Карточка "Создать столбик" присутствует в ленте столбиков
- Добавление неограниченного количества столбиков
- Добавление задач в Inbox
- Перенос задач между столбиками мышкой (drag and drop)
- Перенос задач между столбиками через селект
- Удаление задач
- Отметка задачи как выполненной / невыполненной
- Пустое состояние столбика: "Пусто"
